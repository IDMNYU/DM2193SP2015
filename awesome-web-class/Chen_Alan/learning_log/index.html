Difference between a pseudo class and a pseudo element

Pseudo classes select regular elements under certain conditions such as position relative to other elements or the state of the element. Single colon. Examples: ":visited", ":active", ":nth-child(n)"

Pseudo elements create new elements beyond the document language. Double colon. Examples: "::after", "::before", "::first-letter"

Display: none vs Visibility: hidden

  • Display: none will render the page as if the element didn't exist. Commonly used with javascript.
  • Visibility: hidden will hide the element but still take up the space
